Listas HTML

As listas HTML permitem que os desenvolvedores da web agrupem um conjunto de itens relacionados em listas.

Exemplo:

Uma lista HTML não ordenada:

  • Item
  • Item
  • Item
  • Item

Uma lista HTML ordenada:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item
  4. Quarto item

Lista de descrição HTML

HTML também oferece suporte a listas de descrição.

Uma lista de descrição é uma lista de termos, com uma descrição de cada termo.

A <dl> tag define a lista de descrição, a <dt> tag define o termo (nome) e a <dd> tag descreve cada termo:

Exemplo:

Coffee
- black hot drink
Milk
- white cold drink

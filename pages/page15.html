Elementos em Nível de Bloco

Cada elemento HTML possui um valor de exibição padrão, dependendo do tipo de elemento.

Os dois valores de exibição mais comuns são bloco e embutido.

Exemplo:

Um elemento de nível de bloco sempre começa em uma nova linha e os navegadores adicionam automaticamente algum espaço (uma margem) antes e depois do elemento.

Um elemento de nível de bloco sempre ocupa toda a largura disponível (estende-se para a esquerda e para a direita tanto quanto possível).

Aqui estão dois elementos de bloco comumente usados em HTML:

<p>Hello World</p>

<div>Hello World</div>

Elementos de Bloco em HTML:

<address>, <article>, <aside>, <blockquote>, <canvas>, <dd>, <div>, <dl>, <dt>, <fieldset>, <figcaption>, <figure>, <footer>, <form>, <h1> - <h6>, <header>, <hgroup>, <hr>, <li>, <main>, <nav>, <noscript>, <ol>, <p>, <pre>, <section>, <table>, <tfoot>, <ul>, <video>.

Elementos Embutidos

Um elemento embutido não começa em uma nova linha e ocupa apenas a largura necessária.

Aqui estão os elementos embutidos em HTML:

<a>, <abbr>, <acronym>, <b>, <bdo>, <big>, <br>, <button>, <cite>, <code>, <data>, <dfn>, <em>, <i>, <img>, <input>, <kbd>, <label>, <map>, <object>, <q>, <samp>, <script>, <select>, <small>, <span>, <strong>, <sub>, <sup>, <textarea>, <time>, <tt>, <u>, <var>.

O Elemento <div>

O <div> elemento é frequentemente usado como contêiner para outros elementos HTML.

O <div> elemento não possui atributos obrigatórios, mas style, class e id são comuns.

Quando usado junto com CSS, o <div> elemento pode ser usado para estilizar blocos de conteúdo.

Exemplo:

London

London is the capital city of England. It is the most populous city in the United Kingdom, with a metropolitan area of over 13 million inhabitants.

O Elemento <span>

O <span> elemento é um contêiner embutido usado para marcar uma parte de um texto ou parte de um documento.

O <span> elemento não possui atributos obrigatórios, mas style, class e id são comuns.

Quando usado junto com CSS, o <span> elemento pode ser usado para estilizar partes do texto.

Exemplo:

My mother has blue eyes and my father has dark green eyes.

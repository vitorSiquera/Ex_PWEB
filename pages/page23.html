Entidades de caracteres HTML

Alguns caracteres são reservados em HTML.

Se você usar os sinais de menor que (<) ou maior que (>) em seu texto HTML, o navegador poderá misturá-los com tags.

Nomes ou números de entidades podem ser usados para exibir caracteres HTML reservados.

Os nomes das entidades são assim:

&entity_name;

Os números das entidades são assim:

&#entity_number;

Para exibir um sinal de menor que (<) devemos escrever: &lt; ou &#60;

Os nomes das entidades são mais fáceis de lembrar do que os números das entidades.

Espaço inquebrável

Uma entidade HTML comumente usada é o espaço inseparável:   (&#160;)

Um espaço ininterrupto é um espaço que não será dividido em uma nova linha.

Outro uso comum do espaço ininterrupto é evitar que os navegadores trunquem espaços em páginas HTML.

Se você escrever 10 espaços no seu texto, o navegador removerá 9 deles. Para adicionar espaços reais ao seu texto, você pode usar o   entidade de personagem.

O hífen inseparável ( ‑ ) é usado para definir um caractere de hífen (‑) que não se divide em uma nova linha.

Combinando marcas diacríticas

Uma marca diacrítica é um “glifo” adicionado a uma letra.

Algumas marcas diacríticas, como grave (̀) e aguda (́) são chamadas de acentos.

Marcas diacríticas podem ser usadas em combinação com caracteres alfanuméricos para produzir um caractere que não esteja presente no conjunto de caracteres (codificação) usado na página.

Aqui estão alguns exemplos:

Mark Character Construct Result
̀ a &#768; à
́ a &#769; á
̂ a &#770; â
̃ a &#771; ã
̀ O &#768; Ò
́ O &#769; Ó
̂ O &#770; Ô
̃ O &#771; Õ
